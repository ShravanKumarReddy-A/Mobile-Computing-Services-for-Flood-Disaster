# 🌊 Mobile Computing Services to Support Flood Disaster App

An Android-based mobile application developed to improve flood disaster management and relief coordination through real-time communication between users, donors, supporters, and administrators. The system helps streamline aid requests, verification, and resource allocation during flood emergencies. 

---

## 📌 Project Overview

This project focuses on building a centralized disaster management platform that enables affected users to request help and allows donors and supporters to provide assistance efficiently.

The application improves:

* Real-time communication
* Resource coordination
* Request verification
* Disaster response transparency
* Faster aid distribution

The system is designed using Android and Kotlin with a backend database for storing user and request information. 

---

## 🎯 Objective

The main objective of the project is to develop a mobile application that supports flood disaster relief operations by:

* Providing a centralized communication platform
* Managing requests efficiently
* Ensuring verified aid distribution
* Improving disaster response speed
* Enhancing coordination among all stakeholders



---

## ❗ Problem Statement

Traditional disaster management systems rely heavily on manual coordination and fragmented communication, which often results in:

* Delayed aid delivery
* Inefficient resource allocation
* Lack of transparency
* Difficulty verifying requests
* Poor real-time communication

This project solves these challenges using a mobile-based digital solution. 

---

## 🚀 Proposed Solution

The proposed system is an Android application that allows:

### 👤 Users

* Register/Login
* Raise flood assistance requests
* Track request history
* Access flood safety information

### 🤝 Supporters

* Register/Login
* View requests
* Accept and support requests

### 💰 Donors

* Register/Login
* View verified requests
* Provide donations/support

### 🛡️ Admin

* Verify supporter profiles
* Verify user requests
* Monitor platform activities
